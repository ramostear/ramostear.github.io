JavaI/O
  • Java

1、基于ant.jar的文件操作

1-1 解压压缩文件

这里使用ant.jar来进行文件的操作，文件的压缩格式为.ZIP

使用结束后需要把打开的所有I/O流关闭，否则会造成JVM占用该文件，从而不能够删除所操作的文件，需要重新启动计算机或者服务器才能够进行修改和删除操作。重点需要关注的是ZipFile流是否正常关闭。

1-2 删除文件

删除某个文件夹下的所有文件，使用递归删除的方式，从文件夹最里面的文件开始进行删除，首先判断当前文件夹下是否有文件存在，如果没有，直接删除该文件夹，如果有文件存在，这获取该文件夹下的所有文件，递归依次进行文件的删除，最后还需要删除最外层的文件夹。
